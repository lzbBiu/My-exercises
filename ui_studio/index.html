视觉设计(Graphic UI Designer)

视觉设计要从UI设计中提取细节，然后把这些细节通过视觉效果最后传达给用户，同时视觉设计在设计过程中也会对UI设计起到检查和反馈的作用，在完成视觉还原的同时要指导工程师进行应用布局。

UI交互(Interation Designer)

UI设计是指对软件的人机交互、操作逻辑、界面美观的整体设计。好的UI设计不仅是让软件变得有个性有品位，还要让软件的操作变得舒适简单，自由，充分体现软件的定位和特点，将设计贯穿产品始终。

用户体验(User Experience Engineer)

用户体验指的是用户参与的创新，是以用户至上的观点作为基石的。是一种纯主观的在用户使用一个产品或服务的过程中建立起来的心里感受。将其应用到设计当中，以优化产品的体验。
